# Landing Page BBI Auto

Landing page website untuk aplikasi BBI Auto - Aplikasi Digital Bengkel untuk Perawatan Kendaraan.

## Fitur

- ✅ Desain responsive untuk desktop dan mobile (Android)
- ✅ Animasi smooth scroll
- ✅ Menu hamburger untuk tampilan mobile
- ✅ Animasi scroll reveal untuk elemen-elemen
- ✅ Header sticky dengan efek hide on scroll
- ✅ Desain modern dengan color scheme merah-putih

## Teknologi yang Digunakan

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (Vanilla)
- Google Fonts (Poppins)

## Struktur Folder

```
LandingPageBBIAuto/
│
├── index.html          # File HTML utama
├── style.css           # File CSS untuk styling
├── script.js           # File JavaScript untuk interaktivitas
├── logo.svg            # Logo BBI Auto
└── README.md           # Dokumentasi
```

## Cara Menggunakan

1. Buka file `index.html` di browser
2. Website akan otomatis responsive untuk berbagai ukuran layar
3. Untuk testing mobile, gunakan Developer Tools (F12) dan toggle Device Toolbar

## Responsiveness Breakpoints

- Desktop: > 1024px
- Tablet: 768px - 1024px
- Mobile: < 768px
- Small Mobile: < 480px

## Customisasi

### Mengubah Warna
Edit variabel CSS di file `style.css`:
```css
:root {
    --primary-red: #E53935;
    --dark-red: #C62828;
    --light-red: #FFE5E5;
}
```

### Menambah/Mengubah Konten
Edit file `index.html` sesuai kebutuhan.

### Menambah Animasi
Tambahkan class atau modifikasi JavaScript di `script.js`.
